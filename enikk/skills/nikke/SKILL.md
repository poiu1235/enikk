# NIKKE: Goddess of Victory

Play NIKKE: Goddess of Victory using the enikk game controller tools.

## When to Use
- User says "play nikke", "start nikke", "nikke game"
- User wants to automate daily missions, combat, or other NIKKE activities
- User references NIKKE-specific features like Outpost, Campaign, Tribe Tower

## UI Reference

### 大厅界面 (Main Lobby)
1. 妮姬（角色管理）界面
2. 战役编队界面
3. Home键，从其他界面返回大厅
4. 物品管理界面
5. 招募（抽卡）页面
6. 付费商店
7. 综合兑换店
8. 前哨基地
9. 查看防御等级&收菜
10. 方舟（各种日常玩法界面）
11. 主线大地图
12. 限时活动页面（活动开启后出现按钮）
13. 领取和查看每日、每周任务、主线任务、成就等
14. 与13相同，打开任务界面
15. 好友界面（管理好友与发送友情点数）
16. 联盟系统
17. 点眼睛可隐藏大厅所有UI
18. 系统设置、邮件领取、官方公告查看等
19. 玩家个人设置界面
20. 活动轮播图，点击可直接跳转到相应界面
21. BalaBala通讯界面、活动列表和其他特殊玩法按钮（EVENT图标有红点说明有活动未完成）

### 角色管理界面
1. 用角色名搜索角色
2. 切换筛选爆裂1/2/3阶角色
3. 筛选有红点的角色（装备可更新或抽到备用躯体时）
4. 筛选排序角色
5. 跳转到咨询界面

### 角色详情页
1. 更换角色服装
2. 管理收藏品
3. 重置等级，消耗10珠宝，返还所有升级资源
4. 点击好感等级跳转到咨询页面
5. 升级&极限突破（角色放入同步器则升级按钮失效）
6. 查看角色所处部队其他队员的图鉴
7. 调整角色装备（后期在此洗装备词条）
8. 查看、强化角色技能
9. 调整角色魔方

### 收藏品/珍藏品界面
首次点开角色收藏品管理：
1. 绑定收藏品：选择已有收藏品与角色永久绑定（根据武器种类固定类型）
2. 合成收藏品：R级合成随机盒子，SR级合成自选盒子

绑定后：
1. 确认收藏品技能
2. 更换收藏品：R级换SR级，继承经验值（15级R转5级SR）
3. 保养强化收藏品：5/10/15级三阶段，有概率非常成功直接升满
4. 合成收藏品

珍藏品条件：
1. 角色好感度30级（需突破2星）
2. 角色获得SR级收藏品并升满15级
3. 剧情任务走完后收藏品进化为珍藏品

珍藏品保养：
1. 确认珍藏品技能变更
2. 保养珍藏品：使用角色专属材料，160个材料升满3级

### 战役编队界面
1. 切换5个队伍
2. 头像左下角查看角色技能
3. 头像右下角跳转角色详情页
4. 自动编队

### 前哨基地界面
1. 战术学院（解锁各种重要功能）
2. 同步器（管理同步器角色，5名角色200级后继续强化）
3. 进入玩家房间
4. 循环室（消耗碎片强化各职业各企业角色）
5. 强化基础核心（使用主线和成就赠送的核心芯片）
6. 查看防御等级&收菜
7. 派遣探索&派遣收菜
8. 查看支线剧情（每周可看数量取决于体力点数）

### 指挥官房间界面
1. 回看主线、支线剧情、档案馆入口
2. 查看主线光点拾取的遗失物品，领取收集奖励（珠宝）
3. 欣赏、设置游戏BGM，领取收集奖励（珠宝）
4. 妮姬咨询入口

### 咨询界面
1. 关注角色（在咨询列表置顶）
2. 查看角色个人剧情
3. 送礼
4. 进行咨询对话
5. 查看好感度追加能力值

### 防卫收菜界面
1. 查看当前装备掉落率
2. 查看累计时间、容量、当前等级的资源恢复量
3. 消耗珠宝立即获得2小时收菜量（每日首次免费）
4. 点击收菜

### 方舟界面
1. 迷失地区（一次性迷宫，产出魔方装备）
2. 无限之塔（每日，可勾选自动连续爬塔）
3. 模拟室（每日，产出技能强化书）
4. 竞技场（每日，PVP）
5. 拦截战（每日，产出装备和洗词条石头）
6. 成长和进度排名（红点可领取排名奖励）

### 迷失地区界面
1. 选择迷宫关卡
2. 领取探索度奖励
3. 重置迷宫（不重置奖励）
4. 开始挑战迷宫
5. 强化、管理魔方

### 模拟室界面
1. 选择模拟室等级并开始挑战（每天仅需挑战最高难度）
2. 查看当前模拟室BUFF（每次可带走一个BUFF，最多8个）

### 拦截战界面
1. 查看战斗奖励
2. 查看BOSS信息
3. 模拟练习（不消耗门票，不计成绩）
4. 复制前一次战斗结果进行扫荡
5. 正式战斗（放弃或团灭消耗门票计入成绩，可反复重试或强退）

### 新人竞技场界面
1. 查看交战记录
2. 跳转到竞技场兑换店
3. 当前竞技场货币量
4. 编辑竞技场队伍（注意编辑防守队伍）
5. 用珠宝购买额外入场券
6. 刷新对手（免费，无限次数）
7. 查看赛季奖励内容
8. 查看赛季排名

### 特殊竞技场界面
1. 当前竞技场货币量
2. 特殊竞技场积分（红币）
3. 查看交战记录
4. 跳转竞技场兑换商店
5. 点击领取竞技场货币和积分（竞技场收菜）
6. 编辑队伍（需分别安排进攻战、防守战的1/2/3三队）
7. 用珠宝购买额外入场券
8. 刷新对手（免费，无限次数）

### BalaBala聊天界面
1. 支线任务相关聊天
2. 角色单独聊天（抽到新角色必触发聊天且奖励100好感度）
3. 群聊（主线、活动剧情相关群聊）

### 好友界面
1. 搜寻、添加好友
2. 排序好友
3. 一键赠送和领取友情点数
4. 处理收到的好友申请
5. 管理已加好友（删除好友）

### 联盟界面
1. 退出联盟（联盟突袭期间不可退出）
2. 查看联盟成员
3. 查看联盟日志或留言
4. 联盟射击场
5. 联盟突袭玩法
6. 跳转到联盟商店

### 联盟射击场界面
1. 选择属性靶场
2. 选择靶场时间（最右侧是3分钟BOSS战）
3. 查看成绩排名
4. 查看靶场成绩列表
5. 开启或关闭属性固定（固定成员变为200级，不计入成绩）

### 联盟突袭界面
1. 查看BOSS血量
2. 查看BOSS弱点和技能
3. 刷新BOSS血量
4. 查看联盟队友出刀记录
5. 编辑队伍
6. 模拟战练习（不计入成绩，不扣次数）
7. 正式战斗（团灭或放弃扣除次数计入成绩）

### 单人突击界面
单人突击使用25名角色组成5支队伍轮流与BOSS作战。
每期分为前7级BOSS和最后的挑战BOSS。
前7级BOSS：五支队伍合力击杀获得奖励并晋级。
挑战BOSS：无血量，队伍固定400级，时间用尽或团灭后结算，5队打完结算总排名。
前7级BOSS每日3次入场，挑战BOSS每天3次入场（分别计算）。

1. 切换BOSS等级
2. 点击放大镜查看BOSS详细技能和弱点
3. 查看击杀奖励（首次击败奖励和反复击败奖励）
4. 模拟练习（不消耗次数，不计入成绩）
5. 快速战斗（击败某等级BOSS后可直接扫荡同等级）
6. 正式战斗（点击即消耗次数进入战斗准备）

### 单人突击战斗准备界面
1. 确认BOSS血量
2. 查看BOSS技能和弱点
3. 放弃战斗（前7级不算过关，挑战BOSS按已完成队伍结算）
4. 我的记录
5. 编队（5个队伍轮流战斗）
6. 正式战斗（团灭/时间用尽/放弃时结算当前队伍，换下一队）

### 排名界面
1. 查看挑战成绩和当前排名
2. 查看排行榜前50名队伍详情，可复制配队

### 抽卡界面
1. 确认抽卡资源（钻、彩票、蓝票、黑票、金票数量）
2. 跳转到黑票、金票兑换商店
3. 编辑愿望单（普池专用）

### 玩家详情页界面
1. 调整背景图片
2. 调整玩家称号
3. 更改玩家名称（需改名券或珠宝）
4. 复制玩家ID
5. 调整玩家头像和边框
6. 填写自我介绍
7. 确认所属联盟
8. 调整代表部队（展示用）
9. 调整展示边框
10. 切换前哨基地练度详情页

### 活动页界面（14天小活动）
1. 查看活动门票数量
2. 查看活动积分
3. 挑战活动BOSS
4. 查看领取活动任务奖励
5. 活动积分兑换商店
6. 活动剧情回看
7. 进入活动关卡（底部可查看加成奖励妮姬）

### 活动页界面（21天大型活动）
大型活动自带地图，每天刷新掉落物光点，功能可直接走过去互动或点击下方快捷键自动寻路。
1. 进入活动关卡（分Story1和Story2，分批开放，加成妮姬不同）
2. 挑战活动BOSS
3. 小游戏（每期不同玩法）
4. 签到奖励（14天签到）
5. 活动积分兑换商店
6. 活动每日任务
7. 活动大地图（可放大或查看掉落物收集情况）
